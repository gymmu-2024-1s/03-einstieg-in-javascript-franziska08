Aufgaben für Bedingungen und Schleifen

Sortierverfahren

Bubblesort:

Zuerst muss der Text in den ASCII Code umgewandelt werden. Durch die Übersetztung in den ASCII Code kann die Reihenfolge der Buchstaben bestimmt werden. Wenn die Buchstaben noch nicht stimmen, werden die Buchstaben vertauscht, damit sie in die richtige Rheinfolge kommen.

Selectionsort:

Beim Selectionsort gibt es einen sortierten und unsortiereten Teil. Das kleinste Element wird genommen und an den Anfang der Liste gestellt. Dies geht immer so weiter bis es in einer Rheinfolge ist.

Radixsort:

Es sortiert die Zahlen, indem es diese schrittweise nach ihren Ziffern ordnet, Partitionierungsphase, Zehnersortierungsphase, Zehnerstellen, Hunderterstellen sortieren. So werden die Zahlen am Ende vollständig sortiert und in eine Reihenfolge gebracht. Im Anschluss in der Sammelphase: werden die Zahlen vollständig in eine Reihenfolge gebracht z.B 003, 041,093
